# wdenoise
Wavelet Denoising in ANSI C using empirical bayes thresholding and a host of other thresholding methods. _[Documentation is available here](https://github.com/rafat/wdenoise/wiki)_

## Dependencies

Git and CMake

## Getting Started
```
Clone the project.
cd to directory
cmake .
make
```
